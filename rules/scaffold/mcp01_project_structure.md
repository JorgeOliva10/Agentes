# mcp01_project_structure.md

Ensure that every MCP scaffold generated follows a consistent and extensible directory structure.

## Guidelines

- The project must contain the following directories:
  - docs
  - agents
  - tools
  - src
  - mcp
- Each directory must be created if it does not exist.
- The structure must remain simple and easy to extend.
- Do not create additional directories unless they are necessary.
- The scaffold must work as a base for future MCP agents and tools.
