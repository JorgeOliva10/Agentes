# mcp03_required_files.md

Ensure that the MCP scaffold includes all required base files.

## Guidelines

- The project root must include:
  - README.md
  - MCP_SPEC.md
  - PROJECT_STRUCTURE.md
- The docs directory must include:
  - docs/architecture.md
  - docs/agents.md
  - docs/roadmap.md
- The agents directory must include:
  - agents/registry.md
- The tools directory must include:
  - tools/tools.md
- The src directory must include:
  - src/main.py
- The mcp directory must include:
  - mcp/mcp_config.json
