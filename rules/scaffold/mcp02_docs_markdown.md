# mcp02_docs_markdown.md

Ensure that all documentation generated for the MCP project uses Markdown format.

## Guidelines

- All documentation files must use the .md extension.
- Documentation must include clear headings and sections.
- Avoid generating documentation in other formats such as .txt, .html, .rst or .pdf.
- Use Markdown headings, lists and sections to structure the information.
- Documentation must be readable and organized.
