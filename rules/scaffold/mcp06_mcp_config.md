# mcp06_mcp_config.md

Ensure that the MCP project contains a base MCP configuration file.

## Guidelines

- The configuration file must be located at:
  - mcp/mcp_config.json
- The configuration must include:
  - name
  - version
  - agents
  - tools
- The configuration should represent a minimal valid MCP configuration.
- Agents and tools can initially be empty lists.
